Prerequisite
----------------------------------------
Tomcat 8.5
JBPM and KIE 7.0.0


Installation files directories
----------------------------------------
./lib - Libraries
./config - config files

----------------------------------------

jbpm-console.war
kie-tomcat-integration-7.0.0.Final.jar

o bitronics jars
btm-2.1.4.jar
btm-config.properties
btm-tomcat55-lifecycle-2.1.4.jar

o Datasource and transactions jars
h2-1.3.173.jar
jta-1.1.jar
mysql-connector-java-5.1.18.jar
